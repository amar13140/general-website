# Responsive Portfolio Website

A modern multi-page personal portfolio website built using semantic HTML5, advanced CSS3, Flexbox, CSS Grid, and responsive web design principles.

## Features

* Semantic HTML5 structure
* Responsive design for mobile, tablet, and desktop
* CSS Grid for project layouts
* Flexbox for navigation and alignment
* Accessibility-friendly structure
* SEO optimized meta tags
* Dark/Light mode toggle
* Interactive hover effects
* Accessible contact form
* Mobile-first responsive architecture
