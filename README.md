# Personal Portfolio Website

A fully accessible multi-page personal portfolio website built using modern semantic HTML5 and WCAG accessibility standards.

---

## Project Overview

This project is a semantic and accessible personal portfolio website designed to follow modern web development best practices.

The website includes multiple pages, responsive layout support, SEO optimization, keyboard navigation accessibility, and screen reader support.

---

## Features

- Semantic HTML5 structure
- WCAG accessibility support
- ARIA labels for screen readers
- SEO-friendly meta tags
- Responsive design
- Keyboard accessible navigation
- Accessible contact form
- Lighthouse optimized structure

---

## Pages Included

- Home Page
- About Page
- Projects Page
- Contact Page

---

## Technologies Used

- HTML5
- CSS3
- GitHub Pages
